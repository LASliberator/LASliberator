lasliberate
===========

a simple command-line tool to free LiDAR locked-up in ESRI's "LAZ clone"

usage:
lasliberate locked_up.zlas set_free.las
lasliberate imprisoned.zlas liberated.laz




lasliberateGUI
==============

a simple GUI to free LiDAR locked-up in ESRI's "LAZ clone"

usage:
lasliberateGUI
