lasliberate
===========

a simple tool to free LiDAR locked-up in ESRI's "LAZ clone"

usage:
lasliberate locked_up.zlas set_free.las
lasliberate imprisoned.zlas liberated.laz




lasliberateGUI
==============

a very simple GUI to free LiDAR locked-up in ESRI's "LAZ clone"

usage:
lasliberateGUI
